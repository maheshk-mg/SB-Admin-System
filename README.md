# SB Admin System

Angular 19 admin dashboard with Angular Material and SCSS.

## Prerequisites

- Node.js 22+
- npm 10+

## Setup

```bash
npm install
```

## Development

```bash
npm start
```

Open [http://localhost:4200](http://localhost:4200) in your browser.

## Build

```bash
npm run build
```

Output is in the `dist/` folder.
